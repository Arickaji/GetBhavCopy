Features

Everything you need

Built for traders who need reliable NSE data fast, every day.

↓

Equity & Indices

Downloads both NSE equity and index bhavcopy data in one click — no manual steps.

⇄

Symbol Mapping

Rename any NSE symbol or index to match your tool's naming convention. Saved automatically.

⚡

Parallel Downloads

Uses up to 8 concurrent threads for fast multi-day downloads. Weeks of data in seconds.

◑

Dark & Light Theme

Follows your system theme automatically. Manual toggle to override — dark, light or system.

⊞

TXT & CSV Export

Export in TXT or CSV format. Compatible with AmiBroker, MetaStock and most charting platforms.

✓

Skip Existing

Already downloaded a date? It gets skipped automatically. Re-running is always safe.

Installation

Get started in seconds

Requires Python 3.10 or higher.

git clone https://github.com/AricKaji/GetBhavCopy.git
pip install -e .
python -m getbhavcopy

On Windows, use pip install -e . inside a virtual environment for best results.
